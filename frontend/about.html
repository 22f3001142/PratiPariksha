About PratiPariksha

PratiPariksha is a modern CBT platform designed to transform examinations into intelligent, fair, and student-centric learning experiences.

Features:

  • Full CBT exam system
  • Student analytics
  • Gamified learning
  • Teacher insights using IRT
  • AI learning assistant
  • Transparent testing

Vision:

To create a stress-aware and intelligent examination platform.
